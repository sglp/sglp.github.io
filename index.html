Simple Location Project

This is a simple location project, which contains a client, written in PhoneGap (and thus, being able to be deployed to many mobile platforms) and a server, written in pure PHP.

Unfortunately, both client and server has nasty bugs placed so deeply in core code, that to fix it, entire project would actually have to be rewritten from scratch. I don't have time to do so and I don't plan to do it. That is, why you should treat this project just as a toy, a funny experiment or an introduction to writing your own solution. You shouldn't absolutely use it in production environment or for any kind of serious location or tracking issues.

More information about this project and its both components, you'll find in client and server repositories.
